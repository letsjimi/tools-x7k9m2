# WING Latenz Calculator v3

Mobile-first Latenz-Rechner für die Behringer WING Konsole.

## Datenquelle
Exakte Latenz-Werte aus WING Reference PDF bei 48kHz Sample-Rate.

## Latenz-Werte (bei 48kHz)
- **Standard:** 32 Samples (0.67ms) — die meisten Effekte
- **Speaker Manager:** 33 Samples (0.69ms)
- **Mastering:** 36 Samples (0.75ms)  
- **Exciter:** 43 Samples (0.90ms)
- **Unclear/Frequency dependent:** Dimension Chorus, Stereo Pitch, Combinator, Precision Limiter, Psycho Bass, Pitch Fix, Angel Amp, Tape, Mood Filter, Bodyrez, Rack Amp, Rock Amp, Jazz Clean Amp, Delux Amp

## Features
- Mobile-first, Single Page
- Große Touch-Buttons für alle Effekte
- Sample-Rate Umschaltung (44.1 / 48 / 96 kHz)
- Gesamt-Latenz groß angezeigt (Input + Output + Gesamt)
- Effekt-Rack zeigt aktive Effekte als Chips
- 6 Kategorien: Reverb, Delay, Mod/FX, Dynamics, Pitch/Amp, Channel
- Unclear-Werte kursiv mit gestrichelter Umrandung
- Exception-Werte farblich hervorgehoben
